Reading from a CSV In Idyll

By: Matthew Conlen

I recently added support for loading datasets directly from .csv files, eliminating the need to convert them to json by hand.

data
lake-huron.csv
index.idl
styles.css
package.json

In my data folder I have a file named lake-huron.csv. It contains rows of data about the water level in Lake Huron over time.

To load it, I just have to include a data component and point the source to a .csv file.

[data name:"lakeHuron" source:"lake-huron.csv" /]

I can quickly display it in a table.
